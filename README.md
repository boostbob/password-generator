password-generator
==================

A very simple password generator.

```
gem build password-generator.gemspec
gem install password-generator-0.0.1.gem
```

```ruby
=> true
>> PasswordGenerator.new_password
=> "C9Quq9T,&a:4"
>> PasswordGenerator.new_password
=> "EN@yEq:f+LIW"
>> PasswordGenerator.new_password
=> "N,nP}/m=aaX4"
```
